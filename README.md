# 🛡️ Cybersecurity Educational Platform

Este é um projeto educativo interativo desenvolvido para ensinar os fundamentos da cibersegurança. A plataforma aborda conceitos críticos como tipos de malwares, categorias de hackers e os pilares fundamentais da segurança da informação.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estruturação semântica e acessível.
- **CSS3 Moderno**:
  - Arquitetura de estilos globais e específicos.
  - Interface com conceitos de _Glassmorphism_ (efeito de vidro fosco).
  - Animações `@keyframes` para suavidade visual.
- **JavaScript (ES6+)**:
  - Manipulação dinâmica do DOM para quizzes interativos.
  - Persistência de dados com `localStorage` para contador de visitas.
  - Lógica de eventos para interatividade com o utilizador.

## 🛠️ Funcionalidades Principais

- **Quiz de Malwares**: Teste de conhecimentos sobre ameaças digitais com feedback de pontuação.
- **Exploração de Hackers**: Painel interativo para identificar as diferentes éticas e objetivos dos hackers.
- **Desafio dos Pilares**: Quiz focado nos princípios de Confidencialidade, Integridade e Disponibilidade.
- **Experiência Dinâmica**: Relógio em tempo real e saudações que se ajustam ao horário do utilizador.
